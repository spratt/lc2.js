lc2asm.js Documentation

Project Index

Methods

lc2.tokenize(str)
Splits a string into an array containing an array of tokens for each line in the input string.
lc2.removeComments(lines)
Given an array of arrays (lines) of token strings, strip out any tokens after a semicolon on each line.
lc2.parse(lines)
Given an array containing an array of arrays (lines) of token strings, return an object with key "symbols" whose value is a mapping from symbols to memory locations, and "lines" whose value is the input array with comments stripped out.
lc2.translate(ob)
Given an object with key "symbols" whose value is a mapping from symbols to memory locations, and "tokens" whose value is an array of arrays of token strings, translate the tokens to an object with key "start" whose value is the memory location of the first instruction and "bytecode" whose value is the bytecode string corresponding to the input tokens.
lc2.assemble(str)
Given a string containing LC-2 assembly code, return an object with key "start" whose value is the memory location of the first instruction and "bytecode" whose value is the bytecode string corresponding to the input string.
